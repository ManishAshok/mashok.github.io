Projects

Detection of Zero Day Security Threat using ML

Our method for spotting dormant malware utilized Python and its rich data processing and machine learning libraries, like Scikit-learn and Pandas. We began by assembling and preparing data, extracting features from network and system logs that signal malware presence. A classification model was trained with these features, using varied algorithms to discern malware from benign software effectively. Through iterative development, we continually refined our model with fresh data, aiming to boost precision and minimize false alarms. This project demonstrated the potential of machine learning in identifying security threats using system and environmental variables.

"Paper entitled the same was published in International Journal of Current Engineering And Scientific Research, Volume-8, Issue-9."

Twitter Feed Live mood detection Chrome Extension

For the Live Mood Detection Twitter Feed Chrome Extension, I employed a sophisticated technology stack. The core of the extension was developed using JavaScript, the standard scripting language for Chrome extensions, allowing for dynamic content manipulation within the Twitter feed. HTML and CSS were utilized for crafting the user interface, ensuring a seamless and intuitive experience. The sentiment analysis was powered by the VadarSentiment Python library, which required an integrated approach to connect Python's processing capabilities with the JavaScript-driven extension. This was accomplished through a background script that communicated with a Python server to analyze tweet sentiments in real time.

The choice of tools ensured a robust and responsive extension that enriches user interaction with social media content

Dog Care Provider

In the Dog Care Provider initiative, we capitalized on Spring Framework's capabilities, focusing on its robust ecosystem that encompasses Spring MVC for web request management and Spring Security for advanced authentication and authorization processes. Utilizing Java, we structured server-side operations that effectively respond to user requests. Spring Boot was employed for quick development, minimizing redundant code and prioritizing our business objectives.

We integrated Amazon DynamoDB with the AWS SDK for Java, ensuring scalable and secure data transactions. This streamlined interactions and enhanced user experiences, making our application more responsive and secure.

Smart Inventory Management

The creation of our Smart Inventory Management system involved Python and Django for backend operations, with PostgreSQL for database needs. Django's ORM was crucial for smooth database interactions, and we developed a user-friendly frontend with HTML, CSS, and JavaScript. A significant hurdle was real-time inventory updates, which we surmounted by incorporating Django Channels and WebSocket for asynchronous data communication.

Following Agile principles, our development consisted of sprints to address specific functionalities, allowing for adaptability and continuous enhancement. The outcome was a dynamic, user-centric inventory system that efficiently handled data.
